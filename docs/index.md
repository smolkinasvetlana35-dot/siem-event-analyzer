# SIEM Event Analyzer

Система анализа событий безопасности с использованием именованных кортежей (namedtuple).

## Основные возможности

- Генерация тестовых событий с помощью Faker
- Фильтрация событий по уровню критичности
- Группировка событий по типам
- Извлечение уникальных IP-адресов
- Демонстрация неизменяемости namedtuple
- Генерация аналитических отчетов
- Сериализация и десериализация состояния
